# SNIBruter

**SNIBruter — это Python скрипт, который автоматически проверит и подберет валидные SNI для вашего сервера.**

# Использование

**`python sni_bruter.py IP port --sni-path filename --concurrency number`**

# Аргументы

- **IP — IP-адрес X-UI панели/nginx**
- **port — порт X-UI панели/nginx**
- **filename — файл словаря**
- **number — количество потоков**

# Словари

- **Операторские — публичные и служебные домены операторов**
- **big_list.txt — публичные и служебные домены российского и зарубежного сегментов**
- **whitelist.txt — публичные и служебные домены белого списка (из публичных источников)**
- **x5.txt — публичные и служебные домены X5 Group**
- **edge.txt — публичные и служебные домены Edgecenter**
- **ngenix.txt — публичные и служебные домены Ngenix**
